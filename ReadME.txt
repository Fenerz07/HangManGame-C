Hangmangame est le jeu du pendu.
Il suffit de rentrer un mot , une lettre , une phrase ou même des chiffres.
Ensuite il suffit de deviner le mot en 7 essais qui correspondent chacun à une partie du dessin console du pendu.
Le code ne comporte quasiment pas de commentaires mais il est simple à la compréhension .
Le code est réalisé en anglais .

HangManGame by Fenerz.
